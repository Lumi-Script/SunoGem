# Suno Gem - Based on [Suno Reference](https://github.com/stayen/suno-reference)

Gemini Gem instructions + knowledge for Suno lyrics generation, based on Suno Tag Reference. Stripped down for relevance and LLM compatibility.

## Instructions
* Add gem.md raw text to a new Gemini gem 'instructions' box.
* Upload tag-reference-llms.md as 'knowledge'
* Create the gem
Prompt with context for lyrics and style for best results, optionally also negative style.

## Outputs:
* 1. Suno Prompt Settings
** Style of Music:** * **Title:** A suggested title.
** Advanced Parameters:** Suggested Vocal Gender, Weirdness (%), and Style Influence (%).
* 2.  Lyrics
** Full lyrics with Meta Tags
* 3. Automation Script (Console JS)
* 4. Clean lyrics (close to musixmatch format)
